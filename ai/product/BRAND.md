# Orion brand

Orion is calm mission control: dark navy space, restrained cobalt orientation, precise cyan technical signals, and high-contrast information. It feels futuristic, not noisy or game-like.

## Principles

- Dense enough for repeated scanning, never cramped.
- Glow communicates focus or live technical state; it is not general decoration.
- Cobalt connects the icon to primary actions, selection, and keyboard focus; use it as an accent, not a bright filled surface.
- Cyan is reserved for secondary technical signals and live system detail.
- Green, amber, and red are reserved for working, attention, and blocked states.
- A dirty Git working tree is neutral information, not an amber warning; reserve amber for states that genuinely need attention.
- Surfaces use subtle blue hierarchy instead of stacks of glass cards.
- Motion is short (120–250 ms), functional, and disabled when reduced motion is requested.

## Mark

The icon is a cybernetic ouroboros forming an open `O` on a deep-blue rounded tile. `brand/orion-desktop-icon.png` is the approved transparent source; generated platform assets live in `src-tauri/icons/`, and the browser copy in `frontend/public/orion-icon.png`.

Outside the tile is transparent, matching Menma and Akasha. `brand/sources/orion-icon.svg` is an earlier concept, not the runtime source.
